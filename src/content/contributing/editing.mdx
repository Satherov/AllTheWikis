# Editing

This section will explain how you can modify entries or add some new ones.

## Modifying Entries

On every page there is a `Edit this page` button which will take you to the respective file on GitHub.

From there you can click the edit button and make your changes and then submit a pull request. That's the easiest way to edit something. Of course you can still
edit a local copy and submit a pull request from there.

## Adding Entries

New pages go in the `src/content` folder. The top most folders are the sections you can see in the nav bar.

### Sections

To create a new section, you need to create a new folder in the `src/content` folder, put in your `index.mdx` and `_meta.js` file and mark the index file as a `page`.

### Folders

To create new folders, you simply create the folder in the respective section folder and once again add the `index.mdx` and `_meta.js` file.
Make sure to point to the `index.mdx` file from the `_meta.js` file of the section.

This is different from the section index file. You do **not** add the index file to the `_meta.js` file of the folder!
If you do it **will *not* work!**

### Files

Files dont require any special setup. Simply make a new `.mdx` file, add it to the `_meta.js` file of the folder it lives in and you're done
